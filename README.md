# 智能救援小车视觉系统

这是一个为智能救援小车比赛设计的视觉系统，用于检测和识别红、黄、蓝、黑四种颜色的小球，并根据比赛规则进行目标优先级排序。

## 功能特性

- **多颜色检测**：支持红、黄、蓝、黑四种颜色的小球检测
- **紫色围栏识别**：专门识别紫色围栏，用于确定有效操作区域
- **目标跟踪**：实时跟踪检测到的小球位置
- **安全区检测**：自动识别和检测安全区域，避免误抓取己方区域内的小球
- **黄色球限制规则**：严格执行黄色球（危险球）的特殊处理规则
- **围栏内小球过滤**：只考虑紫色围栏外的小球作为有效目标
- **可视化界面**：实时显示检测结果和最佳目标标记
- **配置灵活**：支持通过JSON配置文件调整颜色阈值和游戏策略
- **阈值调试工具**：提供各颜色专用的HSV阈值调试工具
- **错误处理**：增强的错误处理机制，确保系统稳定运行

## 配置说明

### HSV颜色阈值配置 (hsv_thresholds.json)

该文件定义了四种颜色的HSV阈值范围，用于颜色检测：

```json
{
  "red": [
    {"lower": [0, 100, 100], "upper": [10, 255, 255]},
    {"lower": [160, 100, 100], "upper": [180, 255, 255]}
  ],
  "yellow": [
    {"lower": [20, 100, 100], "upper": [30, 255, 255]}
  ],
  "blue": [
    {"lower": [100, 100, 100], "upper": [130, 255, 255]}
  ],
  "black": [
    {"lower": [0, 0, 0], "upper": [180, 255, 50]}
  ]
}
```

### 游戏策略配置 (game_strategy.json)

该文件定义了队伍颜色、黄色球规则和小球数量配置：

```json
{
  "team_color": "red",      // 己方颜色（red或blue）
  "yellow_rules": {
    "max_count": 2,         // 黄色小球数量
    "cannot_hold_other": true  // 夹取黄色小球时不能同时夹其他小球
  },
  "ball_counts": {
    "red": 4,              // 红色小球数量
    "blue": 4,             // 蓝色小球数量
    "black": 4,            // 黑色小球数量
    "yellow": 2            // 黄色小球数量
  },
  "vision_params": {
    "camera_id": 0,         // 摄像头ID
    "image_width": 640,     // 图像宽度
    "image_height": 480,    // 图像高度
    "detection_confidence": 0.8  // 检测置信度
  }
}
```

## 使用方法

### 运行视觉系统

```bash
cd rescue_vision/src
python main.py
```

### 调整配置

1. **调整颜色阈值**：
   - 使用`text`目录下的专用阈值工具进行实时调整（如`红色阈值.py`、`紫色阈值.py`等）
   - 或直接修改`config`目录下对应的阈值JSON文件
2. **更改队伍颜色**：修改`config/game_strategy.json`中的`team_color`字段
3. **摄像头设置**：所有阈值调试工具的摄像头索引已设置为9，确保使用正确的摄像头设备

## 模块说明

### color_detector.py

颜色检测模块，负责将输入图像转换为HSV颜色空间，并根据配置的阈值检测特定颜色区域。

### ball_tracker.py

小球跟踪模块，负责从颜色掩码中识别圆形物体，并计算小球的位置和大小。

### vision_core.py

视觉核心模块，整合颜色检测和小球跟踪功能，实现：
- 动态安全区检测（红色和蓝色安全区）
- 紫色围栏检测和小球过滤
- 黄色球限制规则检查
- 有效小球过滤和最佳目标选择

### vision_serial.py

串口通信模块，负责与电控系统进行数据交互，发送控制命令和接收反馈信息，支持小球优先级排序和过滤。

### main.py

主程序入口，负责初始化视觉系统并启动运行，协调各模块工作。

### 阈值调试工具

各颜色专用的阈值调整工具，支持实时摄像头预览和HSV参数调整，包括：
- 紫色阈值.py（紫色围栏检测）
- 红色阈值.py（红色小球检测）
- 蓝色阈值.py（蓝色小球检测）
- 黄色阈值.py（黄色小球检测）
- 黑色阈值.py（黑色小球检测）

## 比赛规则支持

- **队伍识别**：通过配置`team_color`参数支持红/蓝两队，系统会自动识别己方和敌方小球
- **安全区避让**：自动检测并避开己方安全区内的小球
- **紫色围栏识别**：专门识别紫色围栏，用于确定操作边界
- **小球数量**：支持比赛指定的小球数量配置（红色4个、蓝色4个、黑色4个、黄色2个）
- **危险球处理**：黄色球（危险球）的特殊处理规则，包括检测是否可以抓取
- **围栏限制**：只考虑紫色围栏外的小球作为有效目标

## 注意事项

1. 首次使用前请确保摄像头已正确连接
2. 根据实际环境调整HSV颜色阈值以获得最佳检测效果
3. 比赛前请确认队伍颜色配置正确
4. 视觉系统运行时，按下'q'键可退出程序
5. 使用串口调试工具前，请确保已安装pyserial库：`pip install pyserial`
